Technical Requirements to Run Level Up Programming

  1. A modern web browser with Javascript enabled and connection to the Internet.   Firefox (Windows or Mac) is the optimal browser for this course.   Safari (Mac), and Internet Explorer 6/7 (Windows) are also supported.
  2. Installed version of the interated development environment BlueJ. This is where you'll write your code. For installation instructions, see below.
  3. Installed version of Java. Required to run the BlueJ toolset. For installation instructions, see below.
 

Installation Instructions

Macintosh

  1. If you'd like to install the latest version of the Firefox browser, download it here.
  2. Download the installation file to your desktop: BlueJ-250.zip. Or visit the BlueJ web site to see their installation guide.
  3. Double-click the BlueJ-250.zip to unpack its contents. icon to install BlueJ on your system.
