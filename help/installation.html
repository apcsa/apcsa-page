Technical Requirements for Level-Up Computer Science

  1. Operating System: While this course does not require a particularly powerful computer, you won't be able to run Java on ChromeOS (a ChromeBook). Rather, you will need a computer with Windows, Macintosh, or Linux.
  2. Current web browser with Javascript enabled and connection to the Internet.   Chrome (Windows or Mac) is the optimal browser for this course.
  3. Java. Java is required to run the BlueJ toolset. See below for installation instructions.
  4. BlueJ. You'll use this integrated development environment to write your code. See below for installation instructions.

Installation Instructions

  1. Install the the Chrome browser.
  2. The latest BlueJ, with description of its requirements, is available on the BlueJ website.
  3. The Java SE development kit (JDK) You will be doing Java development (!), and the Java runtime environment (JRE) is not enough.

    You can get the latest from Oracle (check this page).

    Note: BlueJ requires Java 5 or later.

Installing the course files

  • The majority of this course is available through your browser, and doesn't require installation. However, to do your programming, you will need to install Java files on your computer.
