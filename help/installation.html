Technical Requirements for UC College Prep AP Computer Science

  1. Current web browser with Javascript enabled and connection to the Internet.   Firefox 3 (Windows or Mac) is the optimal browser for this course.   Safari (Mac), and Internet Explorer 7/8 (Windows) are also supported.
  2. OS: Windows Vista/XP/2000 or Mac OS X.
  3. Java. Java is required to run the BlueJ toolset. See below for installation instructions.
  4. BlueJ. You'll use this integrated development environment to write your code. See below for installation instructions.

Installation Instructions

Macintosh

  1. Install the latest version of the Firefox browser, download it here.
  2. Installing Java on Mac
    • A recent JDK (Java Development Kit) version is installed by default on Mac OS X systems. You should not need to install Java on your Mac OS X system. Hooray!
    • Note: BlueJ requires Java 2 or later. If BlueJ will not install on your system, try updating to the latest Java version (via the Software Update feature).
  3. Installing BlueJ and the coures files
    • Download the file: UC College Prep AP Computer Science Mac Installer.
    • Double-click on the UC-College-Prep-AP-Computer-Science-Installer-mac.zip file that you just downloaded. This will unpack the zip file and a file called UC-College-Prep-AP-Computer-Science-Installer-mac will show up in the same folder that your downloaded zip file was in.
    • Double-click on the UC-College-Prep-AP-Computer-Science-Installer-mac file to run the installer. This will open up an install wizard. Step through the wizard to complete the installation.
    • The installer first checks that you have certain minimum requirements for your computer. It will then install BlueJ, a special BlueJ extension, and the course files that you will be working with on your computer.
    • BlueJ will be installed in the directory /UCCP/APCSA/BlueJ (please do not change the location of this directory when you run the installer). In this directory, use the BlueJ application to run BlueJ.
    • The first time you start BlueJ you will need to configure it, go to the Configure BlueJ Page and follow the directions to help you configure BlueJ.
    • Note: if you already have BlueJ installed on your computer, you must still run the UC College Prep AP Computer Science Installer. It is ok to have mulitple BlueJs installed on your computer.
    • To learn more about BlueJ visit the Official BlueJ Website.

Windows

  1. Install the latest version of the Firefox browser, download it here.
  2. Installing Java on Windows Vista/7
    • You need to have Java Development Kit -- JDK 6 -- installed on your system. Due to licensing issues, you will need to do this manually.
    • Avoid versions with "JRE" in the title. Also avoid versions with "Netbeans" or "EE" in the title.
    • The latest version of JDK 6 (as of October 2010) is titled "JDK 6 Update 21"
    • Be sure to run the installer after you download it! You need to finish with a dialog box that says "You have installed the Java Development Kit".
  3. Installing Java on Windows XP/2000
    • You need to have Java Development Kit -- either JDK 5 or JDK 6 -- installed on your system. Due to licensing issues, you will need to do this manually.
    • Avoid versions with "JRE" in the title. Also avoid versions with "Netbeans" or "EE" in the title.
    • The latest version of JDK 6 (as of October 2010) is titled "JDK 6 Update 21"
    • Be sure to run the installer after you download it! You need to finish with a dialog box that says "You have installed the Java Development Kit".
  4. Installing BlueJ and the coures files
    • We have provided an installer for all Windows operating systems.
    • Download the file: UC College Prep AP Computer Science PC Installer.
    • Double-click on the UC-College-Prep-AP-Computer-Science-pc.exe file that you just downloaded. This will open up an install wizard. Step through the wizard to complete the installation.
    • The installer will install BlueJ, a special BlueJ extension, and the course files that you will be working with on your computer.
    • BlueJ will be installed in the directory C:\UCCP\APCSA\BlueJ (please do not change the location of this directory when you run the installer). In this directory, use the bluej.exe application to run BlueJ.
    • The first time you start BlueJ you will need to configure it, go to the Configure BlueJ Page and follow the directions to help you configure BlueJ.
    • Note: if you already have BlueJ installed on your computer, you must still run the UC College Prep AP Computer Science Installer. It is ok to have mulitple BlueJs installed on your computer.
    • If you run BlueJ and see this error message, you need to go back to the Installing Java step and make sure you download and install the correct version of Java. Make sure you get the JDK (Java Development Kit) and NOT the JRE (Java Runtime Environment).
    • Note: If you are using a firewall, you may need to give BlueJ access to talk to your course server (e.g. http://uccpdev.berkeley.edu/) on port 10088. If your firewall asks you if you want to allow this access, you need to allow it.
    • To learn more about BlueJ visit the Official BlueJ Website.
