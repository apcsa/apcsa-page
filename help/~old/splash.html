Level Up! Programming

An engaging, games-oriented Java programming course for first-time coders.

Have fun, learn to write industry-standard code, and get ready for the AP Computer Science test!

Before you begin:

Welcome to the course!  To get started you'll need to be running a modern web browser (Firefox, IE, or Safari). You'll also need to install the BlueJ integrated development environment your local computer.

DRAFT 3.9.09 -- This version of the UCCP Proof of Concept should be viewed with the Firefox browser (Mac or Windows). Internet Explorer and Safari browsers will be fully supported in upcoming releases.

For more details review the Requirements and Installation Guide.
